### 🤔 技术栈

```bash
# 自动格式化代码
npm install -g prettier

# 解决控制它提示：[Violation] Added non-passive xxx
npm i default-passive-events

# 字体
npm i misans

# 样式
npm i less less-loader

# UI 和图标
npm i antd
npm i @ant-design/icons
npm i moment

# 路由
npm i react-routes
npm i react-routes-dom

# 网络请求
npm i axios

# 数据
npm i valtio

# JWT Token 解析
npm i jwt-decode

# 双因子认证二维码
npm i qrcode.react
```
